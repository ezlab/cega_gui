CEGA Overview

CEGA, (Conserved Elements from Genomic Alignments), is a database of conserved vertebrate elements. The aim of this database is to provide acces to precomputed sets of conserved sequences from different species and at different levels of the vertebrate phylogeny. The identification of these conserved elements is based on the delineation of colinear blocks of 5 representative species (including human and one species at the root level of the clade), the multiple sequence alignment of these blocks using a global-local strategy (Mlagan) and the subsequent computation of elements using PhastCons.

Search Parameters

Search

First select the clade level on which to make your query: Vertebrates, Amniotes, Eutheria, Boreoeutheria and Euarchntoglires. You can then select the species in which you are interested in and finally input a genomich interval such as "chr2:100053307-200055635" or a gene name or ENSEMBL ID. You can also access each CEGA element with their own ID.

Results Display

Table

The table displays all CEGA conserved elements for the request. Filters can be applied to see only noncoding elements. Elements of each row can be further investigated by clicking on the row. Alignment and information of the other species are displayed with a screenshot fro this element on UCSC.

Score

The score is the PhastCons score for the element.

Status

The status shows if the element is coding or not. It also shows if it is intergenic or intronic. Some long non-coding RNAs or miRNAs are also snnotated as such.

Chromatin HMM

The chromatin HMM displays the status for 9 cell lines. Show color code and cell line pattern and reference.

DNase

DNase shows DNA accessibility score from ENCODE based on the merging of the score for 9 cell line. The grayscale represents the score percentage, black = 100% and white = 0% or data missing.

TFBS

The TFBS columns shows the number of TFBS that can be bound to that site. From compilation of ENCODE data

Other DB

Other DB column shows the presence of overlapping element in other known database about conserved elements. U is for UCNEbase, V is for Vista Enhancer Database, C is for Condor database
