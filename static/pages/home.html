Welcome to CEGA

CEGA, (Conserved Elements from Genomic Alignments), is a database of conserved vertebrate elements. TThis database provides acces to precomputed sets of conserved sequences from different species and at different levels of the vertebrate phylogeny.

Possible queries:

  • No input displays all elements for this species, example here is for all vertebrate human noncoding elements.
  • Genomic interval such as chr2:100053307-200055635
  • Chromosome name such as chr4
  • Gene name such as hoxd1 displays elements in the gene location +/- 500bp.
  • CEGA element ID such as 2.65
